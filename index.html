Regra da Cascata

O CSS segue a regra da cascata, onde a prioridade depende da especificidade e da ordem do código.

Especificidade

  • Tag: especificidade baixa
  • Classe: especificidade média
  • ID: especificidade alta

Ordem dos Seletores

.primeiro h1 → Seleciona qualquer <h1> dentro de .primeiro.

.primeiro > h1 → Seleciona apenas o filho direto.

.primeiro h1 + h1 → Seleciona o h1 que vem logo depois de outro h1.

Classes secundarias

  1. :first-child → Primeiro filho
  2. :last-child → Último filho
  3. :nth-child(n) → Posição específica
  4. :nth-of-type(n) → Tipo específico

Display Inline e Block

Block: ocupa toda a largura, aceita width/height.

Inline: ocupa apenas o conteúdo, não aceita width/height.
